Allows for integration of FormCraft (a WordPress form builder) with popular email services - MailChimp, AWeber, GetResponse, and Campaign Monitor.

== Description ==

= What is FormCraft? =

[FormCraft](http://codecanyon.net/item/formcraft-premium-wordpress-form-builder/5335056?ref=ncrafts "FormCraft") is a [WordPress form builder](http://codecanyon.net/item/formcraft-premium-wordpress-form-builder/5335056?ref=ncrafts "WordPress form builder") which allows you to make awesome forms and surveys using a GUI, drag-and-drop interface, and allows you to track submissions and form analytics right from your WordPress admin panel.
[Click here](http://codecanyon.net/item/formcraft-premium-wordpress-form-builder/5335056?ref=ncrafts "Click here") to know more about FormCraft

= What does FormCraft Add-On Pack do? =

This pack integrates popular email services (MailChimp, AWeber, GetResponse and Campaign Monitor) with the forms made with FormCraft. This plugin allows you to add to your list of subscribers when users fill out beautiful forms made with FormCraft.

= Is it Easy to Integrate? =

Yes! Just follow the steps:

1. Upload and activate this add-on plugin
2. Go to the Add-On tab in the FormCraft admin
3. Add the API key (if you don't know how to get the key, you can find instructions here)
4. When you make the form, you can add the list id in Form Options (if you don't know how to get the list id, you can find instructions here)
5. Add an email field and check the option 'Add to MailChimp', if you want to add to your MailChimp list

That's it!

Or click [here](http://ncrafts.net/formcraft/docs/integrations/ "here") for a detailed guide to the integration

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the downloaded files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the FormCraft admin menu, and click on Add-Ons
